# Campeonato de futebol de salão

API para organizar os jogos de futebol de salão de uma empresa.

## Ações da API:

1) Cadastro de jogadores:
- Nome completo
- CPF
- Número da camiseta

2) Cadastro de times:
- Nome
- Jogadores (5 no máximo)

3) Cadastro de partida:
- Horário de início
- Horário de término
- Placar final

4) Ranking de classificação
